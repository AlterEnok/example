# Promin Target OS

React + Vite версия сайта.

## Структура

- `src/components` - секции и UI-компоненты
- `src/hooks` - preloader, reveal-анимации, tilt-эффекты
- `src/styles/global.css` - общий визуальный стиль

## Запуск

```bash
npm install
npm run dev
```

После запуска Vite откроет локальный URL вроде `http://127.0.0.1:5173/`.
